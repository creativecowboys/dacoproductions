# Daco Productions

One-page marketing site for Daco Productions LLC — a full-service media and production company.

Built with [Next.js](https://nextjs.org) (App Router) and deployed on Vercel.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Deployment

Connected to Vercel. Pushes to `main` deploy automatically. Domain: dacoproductions.com.
